# cookie-ignorer

a browser extension that removes cookie consent popups and automatically rejects tracking cookies.

## what is this?

a browser extension that removes cookie consent popups and automatically rejects tracking cookies.

this is a fork of [OhMyGuus/I-Still-Dont-Care-About-Cookies](https://github.com/OhMyGuus/I-Still-Dont-Care-About-Cookies) with a redesigned ui.

## original vs. this version
